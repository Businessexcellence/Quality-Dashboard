## **VISUAL DESIGN GUIDELINES**

### **Color Scheme:**
- **Primary**: #3b82f6 (Blue - trust, stability)
- **Success**: #10b981 (Green - growth, positive)
- **Warning**: #f59e0b (Orange - attention, action)
- **Danger**: #ef4444 (Red - urgency, critical)
- **Neutral**: #6b7280 (Gray - data, professional)

### **Typography:**
- **Headings**: Bold, 32-36pt
- **Subheadings**: Semi-bold, 24-28pt
- **Body**: Regular, 18-20pt
- **Captions**: Regular, 14-16pt

### **Icons:**
- Use consistent icon set (FontAwesome or similar)
- Size: 24-32px for emphasis
- Color: Match color scheme

### **Layout:**
- White background (clean, professional)
- Ample white space (not cluttered)
- Max 5-6 bullet points per slide
- Use visual hierarchy (size, color, position)

## **ENGAGEMENT CHECKPOINTS**

**At 2 minutes**: Check if they're nodding (problem resonates)
**At 5 minutes**: Check if they're leaning forward (capabilities interest them)
**At 7:30 minutes**: The live demo is your peak engagement moment
**At 9 minutes**: Check if they're taking notes (ROI matters)
**At 11 minutes**: Check if they're asking when to start (you've won)

**If losing attention at any point:**
- Skip ahead to live demo (Slide 6)
- Jump to CSAT crisis (creates urgency)
- Ask: "What concerns you most about our current quality process?"
