User Manual - Calculation Methodology

üìä How Calculations Are Done

Journey at Glance

Accuracy %: (Total Opportunity Pass / Valid Opportunities) √ó 100

Sample %: (Valid Opportunities / Total Opportunities) √ó 100

Error %: (Total Opportunity Fail / Valid Opportunities) √ó 100

Audit Count: Sum of all "Opportunity Count" values from Parameter_Audit_Count

Valid Opportunities = Total Opportunities - Opportunity NA

Account Summary

Overall Accuracy: (Sum of Opportunity Pass / Sum of Valid Opportunities) √ó 100

Overall Sample: (Sum of Valid Opportunities / Sum of Total Opportunities) √ó 100

Overall Error: (Sum of Opportunity Fail / Sum of Valid Opportunities) √ó 100

Total Accounts: Count of unique accounts with Audit Status = "YES"

Transactional Overview

Audits Number: Total Population - Sum of all "Opportunity Count" values (mirrors Journey at Glance Total Population)

Accuracy: Overall parameter accuracy percentage for the account

Sample %: Valid samples as percentage of total transactions

Error %: Error rate across all transactions

Critical/Non-Critical: Count based on "Stage" field classification

Parameter Performance

Parameter Accuracy: (Opportunity Pass / Valid Opportunities) √ó 100 for each parameter

Sample %: (Valid Opportunities / Total Opportunities) √ó 100

Error %: (Opportunity Fail / Valid Opportunities) √ó 100

Critical Parameters: Parameters where Stage = "CRITICAL"

Non-Critical Parameters: Parameters where Stage = "NON CRITICAL"

RCA & CAPA

Total RCAs: Count of all RCA/CAPA records

Open Status: Status = "OPEN", "IN PROGRESS", or "PENDING"

Closed Status: Status = "CLOSED", "COMPLETED", or "RESOLVED"

Error Type Distribution: Grouped by "Error Type" column

Impact Levels: Classified by "Impact" column

Data Sources

Required Sheets in Base File.xlsx:

  • Account_Summary: Client information with Audit Status
  • Parameter_Audit_Count: Parameter-level audit data with Opportunity fields
  • Transactional_Overview: Transaction-level data
  • RCA_CAPA: Root Cause Analysis and Corrective Actions
  • Projects: Project tracking data (optional)
  • CSAT: Customer Satisfaction data (optional)

Important Notes

  • All percentage calculations exclude "Opportunity NA" values
  • Journey at Glance only shows clients with Audit Status = "YES"
  • Filters apply real-time calculations across all metrics
  • Chart years are extracted from "Financial Year" column in data
  • Missing data fields are treated as 0 in calculations
